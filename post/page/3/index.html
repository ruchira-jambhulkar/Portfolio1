Projects

April 3, 2020

Sentiment Analysis on IMDB movie review dataset using NLP and LSTM

A classifier was built for movie reviews in the IMDB dataset. The Recurrent Neural Network, in particular LSTM, is used to perform sentiment analysis in Keras. The IMDB dataset contains 25,000 positive and 25,000 negative reviews. So, the dataset is completely balanced. Conducted sentiment analysis by using NLP techniques like removing HTML contents, special characters, punctuations, stop words, lemmatization, and vectorization. Parsed the HTML contents using the BeautifulSoup python package.
March 2, 2020

Data Analysis and Visualization of 2020 Irish election results using R

The visualizations are created to show the vote per party in each election, the change in vote per party from 2016-2020, A comparison to the national average for party share of the vote, The change in the vote for the most significant candidates in both elections. The datasets used are in different formats, as they were obtained from different sources. Used the dplyr package functions for data manipulation that includes filling of empty result values, filtering the data, combining the columns, removing the unwanted columns, renaming the columns, inserting the columns.
June 5, 2017

Prediction of closing price of NIFTY in the Indian Financial Stock Market.

A model developed to approximately predict the closing price of NIFTY by applying a data mining approach and investigate and identify the most influential factors of Indian Financial Stock Market prices. The main objective of this model is to help investors plan their future investment opportunities well. The supervised classification algorithm is used to analyze market closing prices. Then the model is created based on sentiment analysis. Computed the past and present data by stream processing RSS feeds of various news websites for analysis and conserved using NoSQL.
